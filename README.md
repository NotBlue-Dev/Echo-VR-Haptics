# Echo-VR-Haptics

## What is that ?

Echo VR Haptics is an API based project, allowing you to use bhaptics products with echo arena

## Compatibility

Curently :
- X40
- X16
- Arms
- Face

## Event Feedback

- STUNNED
- GOAL
- GRAB
- HIT WALL
- BLOCKING
- HEARTBEAT
- POWER HAND
- BOOST
